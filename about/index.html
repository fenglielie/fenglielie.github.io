为什么写博客

2022年，博客这个几十年前的产物，和当下的移动互联网似乎格格不入，为什么要写博客呢？

首先，我希望自由地记录下自己的所学所感，不受平台的限制。而当下众多的平台都不满足我的需求：

  1. 朋友圈/空间：只有社交，不适合社恐人士，而且当下在朋友圈随意发表一些天马行空，不合时宜的言论是不妥当的。
  2. 抖音/b站：视频而非文字社区，娱乐而非知识社区，直接pass。
  3. 微博/知乎：无法完全掌控自己的内容，充斥着广告和无意义的性别对立等内容，存在无意义的交流。
  4. CSDN/博客园：满屏的广告和繁杂的页面，充斥着低质的内容，不如独立博客可以精准地掌握页面的每一处布局和功能。
  5. 公众号：排版和内容管理没有博客自在，博客的所有内容都存在于本地的md文件中，通过Hexo部署到Github Pages上。 并且公众号和博客是完全不同的逻辑：公众号是封闭的集市，博客是开放的孤岛。

其次，我迫切地需要把零乱的知识整理起来，构建自己的知识网络，之前的学习笔记都是用完即弃的，我希望可以把所有内容整理起来。曾经使用过印象笔记，有道云笔记，但是云笔记毕竟没有本地md文件来得自在。直接使用Typora在本地写markdown笔记也遇到了很多问题：本地文件的组织管理太麻烦，不方便查看，Typora需要付费了。

因此，最终下定决心花了一个周末的时间，学习搭建了这个独立博客，用到的工具链包括

  1. Git + GitHub
  2. Node.js + Hexo + Next
