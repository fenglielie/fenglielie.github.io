Fortran学习笔记——练习:对流扩散方程

Posted on 2022-06-27 In code Symbols count in article: 2.2k Reading time ≈ 2 mins.

之前的DG程序都是用C++写的，并且也积累了一些基础库和框架，不想用Fortran重写一遍，现在需要写一个对流扩散方程求解的代码，一维的比较简单，正好作为Fortran的练手。

对流扩散方程

参考文章： Y. Cheng and C.-W. Shu, Superconvergence of local discontinuous Galerkin methods for one-dimensional convection-diffusion equations , Computers and Structures, v87 (2009), pp.630-641.

一维的对流扩散方程如下

\[ \begin{aligned} u_t + f(u,x)_x &= (b(u) u_x)_x\\ u(x,0)&=u_0(x)\\ u(0,t)&=u(2\pi,t)\\ \end{aligned} \]

其中初值\(u_0(x)\)也是\(2\pi\)周期的，\(b(u)\ge 0\)。

LDG方法使用中间变量把对流扩散方程中的高阶导转化为方程组：

\[ \begin{aligned} u_t + f(u,x)_x &= (a(u) q)_x\\ q - g(u)_x &= 0 \end{aligned} \]

其中\(a(u)=\sqrt{b(u)}\)，\(g(u)\)是\(a(u)\)的原函数。

首先给几个文章中算例：

  • 算例一(热方程)：方程为\(u_t=u_{xx}\)，精确解为\(u(x,t)=e^{-t}\sin(x)\)；
  • 算例二：方程为\(u_t+u_x=u_{xx}\)，精确解为\(u(x,t)=e^{-t}\sin(x-t)\)。

为了简化模型，我们使用下面形式的方程，其中常数\(\lambda,\mu^2\)待定。\((\mu \ge 0)\)

\[ u_t + \lambda u_x = \mu^2 u_{xx} \]

加入中间变量后变成方程组

\[ \begin{aligned} u_t + \lambda u_x &= \mu q_x \\ q - \mu u_x &= 0 \end{aligned} \]

在一维均匀网格下，DG半离散格式变为：find \(u_h,q_h \in V_h^k\), s.t. \(\forall v_h, w_h \in V_h^k\)

\[ \begin{aligned} \int_{I_j} (u_h)_t v_h - \lambda \int_{I_j} u_h (v_h)_x + \lambda \left[ \widehat{u}_h^c(x_{j+1/2}) v_h(x_{j+1/2}^-) - \widehat{u}_h^c(x_{j-1/2}) v_h(x_{j-1/2}^+) \right]\\ = -\mu \int_{I_j} q_h (v_h)_x + \mu \left[ \widehat{q}_h(x_{j+1/2})v_h(x_{j+1/2}^-) - \widehat{q}_h(x_{j-1/2})v_h(x_{j-1/2}^+) \right]\\ \int_{I_j} q_h w_h =- \mu \int_{I_j} u_h (w_h)_x + \mu \left[ \widehat{u}_h(x_{j+1/2})w_h(x_{j+1/2}^-)-\widehat{u}_h(x_{j-1/2})w_h(x_{j-1/2}^+) \right] \end{aligned} \]

为了兼容其他格式，我们把第二个公式再次分部积分进行转化，在使用\(q_h \in V_h^k\)近似\(\mu (u_h)_x \in V_h^{k-1}\)时，记

\[ q_h = \mu \left[(u_h)_x + R^w_h + R^e_h \right] \]

其中基于右边界的修正量\(R^e_h\)和基于左边界的修正量\(R^w_h\)分别满足：\(\forall w_h \in V_h^k\)

\[ \begin{aligned} \int_{I_j} R^e_h w_h &= \left[\widehat{u}_h(x_{j+1/2}) - u_h(x_{j+1/2}^-)\right] w_h(x_{j+1/2}^-)\\ \int_{I_j} R^w_h w_h &= \left[\widehat{u}_h(x_{j-1/2}) - u_h(x_{j-1/2}^+)\right] w_h(x_{j-1/2}^+)\\ \end{aligned} \]

对时间直接使用向前欧拉离散(实践中使用Runge-Kutta格式替代)，计算流程为：

  1. 计算初值\(u_h^0\)，n=0
  2. 基于\(u_h^n\)，计算修正量\(R_h^{e,n}\)和\(R_h^{w,n}\)
  3. 计算\(q_h^n = \mu \left[(u_h^n)_x + R^{w,n}_h + R^{e,n}_h \right]\)
  4. 计算下一步的\(u_h^{n+1}\)，\(n:=n+1\)，回到第二步或者结束计算。

剩下三个数值流通量的选取：

  • \(\widehat{u}_h^c(x^-,x^+)\)：对流项的数值流通量，例如LF流通量；
  • \(\widehat{u}_h(x^-,x^+)\)：取作\(u_h(x^-)\)；
  • \(\widehat{q}_h(x^-,x^+)\)：取作\(q_h(x^+)\)。(后两个也可以左右互换)
